# 🛒 HostelCart

HostelCart is a student-focused e-commerce web application designed for hostel students to easily purchase daily essentials such as snacks, study materials, personal care products, room accessories, and tech items.

## 📌 Features

- Home page with featured products
- Product catalog with categories
- Search functionality
- Shopping cart system
- Add to cart functionality
- Quantity increase/decrease in cart
- Toast notification when item is added
- Coupon code support
- Dark/Light theme toggle
- Responsive design
- About page
- Contact page

## 🖥️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts

## 🚀 How to Run

1. Download or clone the repository

```bash
git clone https://github.com/your-username/HostelCart.git
```

2. Open the project folder

3. Run `index.html` in your browser

No installation required.
